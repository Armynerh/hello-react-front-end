### Prerequisites

In order to run this project you need:

- Install npm
- Install React
- Install RTK


### Setup

Clone this repository to your desired folder:

```sh
  cd budget_app
  git clone https://github.com/Armynerh/hello-react-front-end.git
```

```sh
  cd  hello-react-front-end
```
